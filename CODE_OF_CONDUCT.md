# Code of Conduct

This is the Send Guard for Gmail project code of conduct. It applies to participation in Issues and Discussions.

## Expected Behavior

- Be respectful and constructive.
- Focus on technical facts and user safety.
- Avoid sharing confidential or personal information.
- Use synthetic examples when discussing Gmail content.

## Unacceptable Behavior

- Harassment or discriminatory language
- Publishing private information without permission
- Sharing real email content, customer data, credentials, or confidential Gmail screenshots

## Enforcement

Project maintainers may remove comments or discussions that violate this code.

For conduct concerns, open an Issue without sensitive personal data. Suspected security vulnerabilities must follow the confidential process in [SECURITY.md](SECURITY.md), not the conduct-reporting path.
